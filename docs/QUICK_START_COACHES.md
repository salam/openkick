# Quick Start for Coaches

Welcome to OpenKick! This guide walks you through the essentials so you can manage your team's attendance and tournaments with minimal effort.

## How It Works

OpenKick connects to a WhatsApp number that parents already use. Messages from parents are automatically understood and turned into attendance records. You get a dashboard to see who's coming, create events, and manage tournaments — no more scrolling through group chats.

## Getting Started

### 1. Log In to the Dashboard

Open the web app in your browser (your club administrator will give you the URL). Log in with the credentials provided to you.

The dashboard shows:

- **Upcoming trainings** with attendance counts
- **Open tournaments** with registration status
- **Recent messages** from parents

### 2. Invite Parents

Share the club's WhatsApp join link or QR code with your parent group. You can find both in the dashboard under **Team > Invite Parents**.

Parents scan the code or tap the link, send their child's name, and they're in. No app download, no sign-up form.

## Managing Training Attendance

### View Who's Coming

The **Trainings** page shows each upcoming session with a clear breakdown:

- **Green:** Attending
- **Yellow:** Late / uncertain
- **Red:** Absent
- **Grey:** No response yet

### Send Reminders

If some parents haven't responded before the deadline, tap **Send Reminder**. The system sends a WhatsApp message to those who haven't replied.

Automatic reminders are also sent based on the schedule set by your administrator (typically 24 hours before training).

### Manual Overrides

If a parent calls you directly or tells you in person, you can update attendance manually from the dashboard. Tap a player's name and change their status.

## Managing Tournaments

### Create a Tournament

Go to **Tournaments > New Tournament** and fill in:

| Field | What it means |
| --- | --- |
| **Name** | E.g., "Spring Cup Zürich" |
| **Date & time** | When the tournament starts |
| **Location** | Address or venue name |
| **Registration deadline** | After this date, parents can no longer sign up |
| **Min / Max players** | The system alerts you if registrations are below or above these numbers |
| **Attachments** | Upload the tournament PDF (rules, directions, schedule) |

Once saved, parents are notified via WhatsApp with the key details and a link to register.

### Upload a Tournament PDF

You can also share a tournament invitation directly via WhatsApp to the club number. The system reads the PDF and pre-fills tournament details (date, location, rules) for you to review and confirm.

### Team Formation

When registrations close, the system suggests team line-ups based on the number of players. You can review and adjust before sharing the final teams with parents.

### Alerts

You'll get a notification when:

- Registrations drop below the minimum player count
- Registrations exceed the maximum
- A player cancels after the deadline
- The waiting list changes

## Communication

### Broadcast Messages

From the dashboard, go to **Messages > Broadcast** to send a WhatsApp message to all parents in your team. Useful for last-minute changes (cancelled training, location change, etc.).

### Event Links

Each tournament and training has a shareable link. You can post it in your existing WhatsApp group — parents tap the link to see details and respond.

## Tips

- **Check the dashboard before training** for a quick headcount.
- **Set deadlines generously** — parents often respond late. Automatic reminders help.
- **Use the PDF upload** for tournaments — it saves you from typing details manually.
- **Don't worry about message format** — the system handles all the parsing. If it can't understand a message, it asks the parent to clarify.

## Managing Users

In the **Settings** page, scroll down to the **Users** card to see all registered coaches and admins.

- **Invite a coach:** Click **Invite User**, fill in their name and email, and they'll receive an email with a link to set their password.
- **View the team:** The table shows everyone's name, email, and role.

If you're an **admin**, you can also:

- Change a user's role between Coach and Admin using the dropdown
- Trigger a password reset email for any user
